One-sentence: a compact uppercase status capsule for telemetry states (LIVE, BLE, WIFI, GND, NO LOCK, drone tags), colored by meaning.

```jsx
<Badge tone="green" dot glow>LIVE</Badge>
<Badge tone="ble" variant="outline">BLE</Badge>
<Badge tone="amber">GND</Badge>
```

Tones: green/amber/teal/magenta/red/cyan/ble/wifi/muted. `solid` = active signal, `outline` = quiet label. `dot` adds a leading indicator; `glow` adds a colored halo.
