One-sentence: the instrument container surface — charcoal-slate fill, hairline inset, optional Neuropol X amber title bar — for framing telemetry and settings blocks.

```jsx
<Panel title="Airspace" trailing={<Badge tone="green">LIVE</Badge>}>…</Panel>
<Panel scope>…radar readout…</Panel>
```

`scope` swaps the hairline for the green scope outline + glow. `trailing` sits right in the title bar. Hard corners; this is an instrument, not a card-with-rounded-corners web UI.
